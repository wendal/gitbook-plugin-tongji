Badidu Tongji tracking for GitBook
==============

You can use install it via **NPM**:

```
$ npm install gitbook-plugin-tongji
```

And use it for your book with in the book.json:

```
{
  "plugins": ["tongji"],
    "pluginsConfig": {
      "tongji": {
        "token": "YOUR-TONGJI-KEY"
      }
    }
}
```
